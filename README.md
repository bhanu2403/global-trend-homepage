# GlobalTrend Analytics Platform

A responsive, single-page web application landing page designed for both mobile and laptop screens. 

## Features
- **Mobile-First Responsive Design:** Adapts smoothly across various device viewports using CSS Media Queries.
- **Modern Layout Modules:** Built using **CSS Flexbox** for structural layouts and **CSS Grid** for the interactive metrics section.
- **Single Page Application (SPA) Logic:** Implements lightweight JavaScript event listeners to toggle page views smoothly without browser refreshes.

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid Custom Layouts)
- Font Awesome (Vector Icons)
- JavaScript (DOM Manipulation)
